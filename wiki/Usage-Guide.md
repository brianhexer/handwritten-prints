# Usage Guide

## Basic Usage

### 1. Enter Text
- Type or paste your text in the **Input** section on the left
- The preview updates in real-time
- Use **Ctrl+A** to select all, or manually select text

### 2. Choose Font
- Click the **Handwriting Font** dropdown
- Select from 24+ fonts
- Font preview updates immediately

### 3. Select Language (Optional)
- Choose a language font (Hindi, Tamil, Arabic, etc.)
- Text automatically translates to that language
- Toggle **Auto-transliterate** off to keep original text

### 4. Customize
- **Font Size**: Adjust thickness (0-30pt)
- **Ink Color**: Blue, Black, or Red
- **Letter Spacing**: Add space between letters
- **Word Spacing**: Add space between words
- **Vertical Position**: Move text up/down
- **Paper Margin**: Show/hide margin lines
- **Paper Lines**: Show/hide guide lines

### 5. Generate Image
- Click **Generate Image** button
- Wait for processing (progress shown at bottom)
- Image appears in the Output section

### 6. Export
- **Download as PNG**: Right-click image ‚Üí Save As
- **Download as PDF**: Click "Download as PDF" button
- **Copy**: Right-click image ‚Üí Copy Image

## Language Features

### Auto-Translation
When you select a language font:
1. A dialog appears asking to translate
2. Click "OK" to translate entire document
3. Text translates from English to selected language

### Auto-Transliterate
For character-level script conversion:
1. Enable **Auto-transliterate** toggle
2. Type English characters
3. Characters map to target script in real-time

### Refresh Translation
- Click the **refresh (‚Üª)** icon next to the toggle
- Re-translates from original English text
- Useful if translation needs updating

## Drawing (Beta)

1. Click **Draw (Beta)** button
2. Sketch directly on the canvas
3. Click X to close drawing panel
4. Your drawings appear in the output

## Troubleshooting

**Q: Translation not working**
- Check internet connection
- Try refreshing the page
- Use different source language

**Q: Font not rendering correctly**
- Try a different font
- Check if language matches font
- Clear browser cache

**Q: Image quality too low**
- Increase resolution setting (4x = highest quality)
- Use high DPI on your device
