# Languages & Fonts

- **Fonts**: Kruti-dev, Kalam
- **Script**: Devanagari
- **Features**: Auto-translation, transliteration
- **Language Code**: `hi`

### Tamil (தமிழ்)
- **Fonts**: Kavivanar, Anek Tamil
- **Script**: Tamil
- **Features**: Auto-translation, transliteration
- **Language Code**: `ta`

### Arabic (العربية)

- **Script**: Arabic
- **Features**: Auto-translation, transliteration
- **Language Code**: `ar`

### Chinese (中文)
- **Fonts**: Ma Shan Zheng

- **Features**: Auto-translation, transliteration
- **Language Code**: `zh-CN`

### Japanese (日本語)

### Korean (한국어)

## Font Loading

### Google Fonts (Used)
- Primary source for most fonts
- Supports 7 language scripts
- Automatic fallback to Latin

### Local Fonts (Option)
- Upload custom TTF/OTF files
- Personal handwriting fonts
- Business-specific fonts

## Character Support

### Transliteration Coverage
Each language includes:
- **A-Z** (26 letters)
- **0-9** (10 digits)
- **Punctuation**: . , ! ? : ; - ( )

### Special Characters
- Spaces preserved
- Line breaks maintained
- Formatting retained where possible

## Translation Coverage

Translates from English to:
- Hindi, Tamil, Arabic, Chinese, Japanese, Korean, Russian
- Uses Google Translate API
- Real-time character-by-character translation
- Fallback to transliteration if translation fails

## Adding New Fonts

1. Submit feature request on GitHub
2. Or upload your own custom font
3. Fonts must be TTF or OTF format
4. Maximum file size: 5MB
